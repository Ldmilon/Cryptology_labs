This section describes the mathematical and cryptographic methods used for the RSA algorithm.isPrime(long long n): Checks if a given number is prime, which is essential for selecting the initial $p$ and $q$ values.gcd(long long a, long long b): Calculates the Greatest Common Divisor using the Euclidean algorithm to ensure the public exponent $e$ is coprime to $\phi(n)$.modExp(long long base, long long exp, long long mod): Implements Modular Exponentiation ($base^{exp} \pmod{mod}$). This is the primary function for both encryption ($C = M^e \pmod n$) and decryption ($M = C^d \pmod n$).modInverse(long long e, long long phi): Finds the modular multiplicative inverse of $e$ modulo $\phi(n)$ using the Extended Euclidean Algorithm to calculate the private key $d$.Main Encryption/Decryption Loop:Encryption: Converts a string into numeric values and encrypts them character-by-character.Decryption: Takes encrypted numeric values and restores the original characters using the private key.
